Modelica.Mechanics.MultiBody.Examples

Examples for the 3D mechanical systems library

Information




This package contains example models to demonstrate the usage of the
MultiBody package. Open the models and
simulate them according to the provided description in the models.




Content



  
PackageDescription

  
Elementary
       Elementary examples to demonstrate various features of the 
           MultiBody library
      
  

  
Loops
       Examples to demonstrate how to model systems with kinematic loops.
           Especially the analytic loop handling is shown.
      
  

  
Systems
       Examples of complete system models that include
           3-dimensional mechanics.
