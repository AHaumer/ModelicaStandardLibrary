Modelica.Math.BooleanVectors.allTrue

Returns true, if all elements of the Boolean input vector are true ('and')


Information




Syntax


allTrue(b);




Description



Returns true if all elements of the Boolean input vector b are true.
Otherwise the function returns false. If b is an empty vector,
i.e., size(b,1)=0, the function returns false.




Example


  Boolean b1[3] = {true, true, true};
  Boolean b2[3] = {false, true, false};
  Boolean r1, r2;
algorithm
  r1 = allTrue(b1);  // r1 = true
  r2 = allTrue(b2);  // r2 = false

Modelica.Math.BooleanVectors.firstTrueIndex

Returns the index of the first true element of a Boolean vector


Information




Syntax


firstTrueIndex(b);




Description



Returns the index of the first true element of the Boolean vector b.
If no element is true or b is an empty vector (i.e., size(b,1)=0) the
function returns 0.




Example


  Boolean b1[3] = {false, false, false};
  Boolean b2[3] = {false, true, false};
  Boolean b3[4] = {false, true, false, true};
  Integer r1, r2, r3;
algorithm
  r1 = firstTrueIndex(b1);  // r1 = 0
  r2 = firstTrueIndex(b2);  // r2 = 2
  r3 = firstTrueIndex(b3);  // r3 = 2
