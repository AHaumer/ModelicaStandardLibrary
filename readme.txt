VehicleDynamics and VehicleTest
===============================

These two CVS modules are used to develop the VehicleDynamics library
and associated test cases, refernce data etc.

VehicleDynamics		The library, should be checked out in
			dymola/Modelica/Library.

VehicleTest		Can be checked out anywhere convenient,
			e.g. dev.

Note: we eventually add personal login identites for each CVS user.
