.ModelicaReference.Operators.'sinh()'

Information

Hyperbolic sine function

Syntax

sinh(u)

Description

Returns the sinh of u, with -∞ < u < ∞. Argument u needs to be an Integer or Real expression.

The sinh function can also be accessed as Modelica.Math.sinh.

Examples

sinh(1)
  = 1.1752011936438
