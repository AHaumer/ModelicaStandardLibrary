.Modelica.ComplexBlocks.ComplexMath.Feedback

Information

This blocks computes output y as difference of the commanded input u1 and the feedback input u2:

    y = u1 - u2;

Example:

     parameter:   n = 2

  results in the following equations:

     y = u1 - u2
